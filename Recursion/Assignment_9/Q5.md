# Question-5: Given an array of integersÂ arr, the task is to find maximum element of that array using recursion.


```
Example 1:
Input: arr = [1, 4, 3, -5, -4, 8, 6];
Output: 8

Example 2:
Input: arr = [1, 4, 45, 6, 10, -8];
Output: 45
```


## Solution


### Optimal Approach


#### Algorithm


- Time Complexity: $O(n)$
- Space Complexity: $O(n)$


#### Code


```javascript
const arr = [1, 4, 3, -5, -4, 8, 6];
const findMax = (max, i) => {
    if (i === arr.length - 1) {
        return max;
    }
    if (arr[i] > max) {
        max = arr[i];
    }
    return findMax(max, i + 1);
};
console.log(findMax(arr[0], 0));
```
